Hola grupo, les dejo la hoja de ruta del sprint 1 :
1.	Confirmación de equipos: Se definirán los roles y responsabilidades de cada miembro del equipo para asegurar una distribución eficiente del trabajo.
2.	Creación de grupos de WhatsApp: Se formarán grupos de comunicación en WhatsApp para facilitar la comunicación rápida y fluida entre los integrantes del equipo.
3.	Creación de espacios en Google Drive: Se organizarán los documentos del proyecto en Google Drive para asegurar que todos los miembros tengan acceso a la información y puedan colaborar de manera efectiva.
4.	Reuniones por Google Meet: Se programarán reuniones en Google Meet para alinear objetivos, discutir avances, y resolver dudas de manera periódica.
5.	Creación de repositorios en GitHub: Se crearán repositorios en GitHub para el manejo del control de versiones del código y garantizar la colaboración y sincronización adecuada del trabajo de desarrollo.
6.	Creación del diagrama de Gantt: Se elaborará un diagrama de Gantt para definir y visualizar las tareas y plazos del proyecto, permitiendo una mejor planificación y control de tiempos.
7.	Herramientas a utilizar: Se definirán las herramientas tecnológicas que se emplearán para la gestión del proyecto, desarrollo del software, control de versiones, entre otras.
8.	Documentación de presentación: Se preparará la documentación para presentar formalmente el proyecto, incluyendo objetivos, plan de trabajo y cronograma.
9.	Diccionario de datos: Se elaborará un diccionario de datos que contenga la descripción detallada de cada uno de los elementos de datos a utilizar en el proyecto.
10.	Análisis exploratorio de datos: Se realizará un análisis exploratorio para identificar patrones, relaciones y características importantes de los datos que se utilizarán durante el desarrollo del proyecto.
